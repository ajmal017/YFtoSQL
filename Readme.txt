#ES

RESUMEN
Esta aplicación obtiene los datos de la bolsa mediante la APP de Yahoo Financial donde extrae todos los tickets o "ticker" 
listado en el archivo "Elementos.txt".
Además, extrae la información de lo que ocurre en el mundo mediante la API de The Guardian.

NOTA IMPORTANTE: Esta aplicación puede ser un poco lenta, sobre todo la primera vez que se ejecuta, debido a que obtiene información
desde el año 2010 hasta la fecha.

#EN

BRIEF
This app get financial and stock data from Yahoo Financial's API. For do this you must enter all tickets name that you want before
execute this software, on the file "Elementos.txt".
Also, it get worldwide news from The Guardian's API.

IMPORTANTE NOTICE:
This app could be very slow, mainly on the firsts executions, and maybe could fail, this is because get and save the data from year 2010
until now. After a few executions, the programs run must be not longer than a one, two or three minuts
